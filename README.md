# ИИ-ассистент для фабрики по изготовлению кухонь

## Описание проекта

### Название:
**ИИ-ассистент для фабрики по изготовлению кухонь**

### Цель проекта:
Разработка и внедрение ИИ-ассистента для повышения эффективности взаимодействия с клиентами на фабрике по изготовлению кухонь. Ассистент должен обеспечивать осмысленное ведение диалога, предоставлять точные ответы на вопросы клиентов, а также автоматизировать процесс записи на встречу и управления расписанием.

### Функциональность:

1. **Осмысленное ведение диалога с помощью нейросети**  
   Ассистент способен понимать контекст общения и предоставлять релевантные ответы, основываясь на базе знаний и обучении через нейросетевые технологии. Это позволяет улучшить качество обслуживания клиентов и сократить время на решение их запросов [[2]].

2. **Ответы на вопросы по базе знаний**  
   Интегрирована система базы знаний, которая содержит информацию о продукции, услугах, ценах, сроках производства и других важных аспектах работы фабрики. Клиенты могут получить быстрые и точные ответы на свои вопросы без необходимости ожидания оператора.

3. **Запись клиента на встречу и добавление встречи в Google Календарь**  
   Ассистент автоматически записывает клиента на удобное для него время и создает событие в Google Календаре, уведомляя обоих сторон о предстоящей встрече. Это упрощает процесс планирования и минимизирует вероятность ошибок в расписании.

### Использованные сервисы:

- **Qwen**: Платформа использовалась для создания системного промпта и формирования базы знаний, что позволило настроить ассистента под специфику фабрики по изготовлению кухонь.
  
- **Savvy (https://suvvy.ai/)**: Сервис для разработки и настройки ИИ-ассистента. Благодаря интуитивному интерфейсу и мощным инструментам, удалось быстро протестировать и внедрить решение в рабочий процесс.

### Преимущества решения:

- **Повышение удовлетворенности клиентов**: Благодаря круглосуточной доступности и быстрому реагированию на запросы, клиенты получают более качественный сервис.
- **Оптимизация рабочего времени сотрудников**: Автоматизация рутинных задач, таких как запись на встречу и управление расписанием, позволяет сотрудникам сосредоточиться на более сложных аспектах работы.
- **Снижение операционных затрат**: Уменьшение необходимости в большом количестве живых операторов для первичного контакта с клиентами приводит к экономии ресурсов.

### Заключение:
Проект направлен на модернизацию клиентского сервиса фабрики по изготовлению кухонь с помощью передовых технологий искусственного интеллекта. Внедрение ИИ-ассистента помогает сделать процесс взаимодействия с клиентами более эффективным, быстрым и удобным как для покупателей, так и для персонала компании.
